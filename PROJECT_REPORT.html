Preface

CareerCraft AI was developed to solve modern recruitment challenges using AI-driven automation. By replacing static builders with real-time scoring, ATS optimization, and visual analytics, the platform empowers candidates to land jobs and recruiters to identify top talent with precision.

The system is built on a modern serverless stack: Next.js (frontend), Firebase (auth/database/storage), and Google Genkit (AI orchestration). This report details the architecture, implementation, and features of the working MVP.

Note: CareerCraft AI is a functional MVP demonstrating the impact of AI on career management and recruitment workflows.

Chapter 5

Implementation & Performance

5.1 Implementation Description

1. Recruiter Dashboard & Analytics

  • Shortlisting: Candidates are persisted to a private subcollection via Firestore writes.
  • Visual Analytics: Uses recharts to perform real-time aggregation of the shortlist.
  • Charts: BarChart for match distribution and PieChart for role breakdown.

2. Admin Subscription Management

  • 30-Day Logic: Calculates expiry intervals using date-fns from planUpdatedAt.
  • Manual Overrides: Admins can modify roles and review payment proof URLs directly.
