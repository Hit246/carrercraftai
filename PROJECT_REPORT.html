Preface

The current hiring market is highly competitive and heavily mediated by Applicant Tracking Systems (ATS), making resume optimization difficult for candidates and shortlisting inefficient for recruiters. Manual review at scale introduces bias and delay. CareerCraft AI was developed to solve this with AI-driven automation and decision support.

The platform replaces static resume builders with real-time resume scoring, ATS optimization, and AI-assisted content generation. For recruiters, it provides bulk candidate matching and a visual analytics dashboard to track the quality of their talent pipeline and manage shortlisted candidates. The system is built using a modern serverless full-stack architecture: Next.js (frontend), Firebase (backend), and Google Genkit (AI orchestration).

Note: CareerCraft AI is a functional MVP demonstrating the impact of generative AI on recruitment efficiency and career development workflows.

Chapter 2

Project Overview

2.1 Overview of Project

CareerCraft AI is a unified platform serving two primary user groups:

  • Candidates: Access tools like the AI Resume Builder, ATS Optimizer, and Job Matcher to improve their hiring chances.
  • Recruiters: Utilize the AI Candidate Matcher to rank resumes, manage a persistent shortlist of talent, and access a Recruitment Analytics Dashboard to visualize candidate quality.

The system features a tiered subscription model (Free, Essentials, Pro, Recruiter) with automated plan activation via Razorpay webhooks and a comprehensive admin panel for subscription oversight.

3.2 Project Objectives

For Job Seekers

  • Build resumes with live-preview and version control.
  • Receive AI feedback on resume strengths and weaknesses.

For Recruiters

  • Persistent Shortlisting: Save top candidates to a private recruiter dashboard.

For Administrators

  • Manage user plans and approve/reject subscription requests.
  • Subscription Management: Monitor billing cycles, track expiry dates (30-day logic), and override user roles.

4.2 DFD (Data Flow Diagrams)

Recruiter Workflow (Level 2)

This flow describes how a recruiter interacts with the matching and analytics engine.

  • 1. Recruiter uploads multiple resumes and a JD.
  • 2. AI Flow ranks candidates and returns scores + justification.
  • 3. Recruiter clicks "Shortlist" to persist a candidate to Firestore.
  • 4. System writes to the shortlistedCandidates subcollection under the recruiter's user document.
  • 5. Analytics Dashboard performs real-time aggregation of this subcollection to render charts.

Chapter 5

Implementation & Results

5.1 Implementation Details

Recruiter Analytics Engine

Implemented using the recharts library. It performs real-time aggregation of the recruiter's shortlisted candidates to display:

  • Average Match Score: Calculated via useMemo from the active shortlist.
  • Score Distribution: A BarChart grouping candidates into High (76%+), Mid (51-75%), and Low (0-50%) match categories.
  • Role Analysis: A PieChart showing the distribution of talent across different job titles the recruiter has screened for.

Admin Subscription Management

A new administrative module that calculates subscription expiry dates (30 days from planUpdatedAt) and provides a UI for manual plan overrides, ensuring seamless support for billing queries and manual activations.

Chapter 6

Conclusion & Future Scope

6.1 Conclusion

CareerCraft AI successfully demonstrates how Generative AI can be integrated into a full-stack SaaS application to automate complex career and recruitment workflows. By focusing on data-driven shortlisting and visual analytics, the platform provides recruiters with actionable insights that manual processes cannot match, while simplifying administration through a robust subscription management system.

6.2 Future Scope

  • AI Mock Interviews: Interactive voice-based interview prep for candidates.
  • Bulk Outreach: Automated email sequences for shortlisted candidates directly from the dashboard.
  • Job Board API Integration: Connecting with platforms like LinkedIn to pull real job listings for the ATS Optimizer.
